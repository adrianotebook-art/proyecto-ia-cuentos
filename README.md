# Generador de Cuentos Infantiles Ilustrados con IA

## Resumen
Este proyecto propone el desarrollo de un generador automático de cuentos infantiles ilustrados utilizando inteligencia artificial. A través de modelos de texto-texto y texto-imagen, se busca crear historias personalizadas acompañadas de imágenes que faciliten la comprensión y estimulen la creatividad en niños.

## Introducción

### Problema
La creación de contenido infantil educativo y atractivo requiere tiempo, creatividad y habilidades de ilustración. Esto limita la disponibilidad de materiales personalizados para niños.

### Propuesta de solución
Se utilizan modelos de IA para generar cuentos y crear ilustraciones automáticamente mediante prompts optimizados.

## Objetivos
- Generar cuentos infantiles automáticamente
- Crear ilustraciones asociadas
- Aplicar técnicas de prompting

## Metodología
Se diseñan prompts específicos para generar contenido textual e imágenes, integrando los resultados en un notebook.

## Herramientas
- ChatGPT
- NightCafe
- Google Colab

## Resultados
Se generan cuentos coherentes junto con ilustraciones acordes al contenido.

## Conclusión
La inteligencia artificial permite automatizar la creación de contenido educativo de forma eficiente.
